useMemo is a React Hook that allows you to memoize (or "remember") the result of a calculation and recompute it only when certain dependencies have changed. 
This is useful for optimizing performance, especially in cases where a component re-renders frequently and performs expensive calculations.

The React useMemo Hook returns a memoized value. (it's like caching a value so that it doesn't need to be recalculated.)

This can improve the performance of the application. There is one more hook in react to improve performance, that is useCalIback hook.

The useMemo and useCallback Hooks are similar. The main difference is:
- useMemo returns a memoized VALUE.
- useCallback returns a memoized FUNCTION.
