React.memo:
React.memo is a higher-order component (HOC) in React that prevents unnecessary re-renders of a child component if its props haven't changed. 
It optimizes performance by ensuring that a child component only re-renders when its props change, even if the parent component re-renders due to its state or props updating.

In the case of your example, if newFn is passed as a prop to the UseCallbackChild component and doesn't change, wrapping the UseCallbackChild component with React.memo ensures it will not re-render unnecessarily when the parent component (UseCallback) re-renders. 
This is particularly useful when the child component is large or has expensive rendering operations.

useCallback:
useCallback is a hook in React that returns a memoized version of a function, which only changes when one of its dependencies changes.
It helps avoid the creation of new function instances on each render. 
When you pass a function as a prop to a child component, and the child component is wrapped in React.memo, useCallback ensures that the function reference doesn't change unless its dependencies change, preventing the child component from re-rendering.
